New Simulation

To create a new simulation:
  1. Pan around the map to the region you are interested in using the pan tool () to find the general area.
  2. Zoom into the specific area of interest using the region zoom tool () to get close enough to see the region
  3. Draw a polygon around the specific feature / features you are interested in using the polygon selection tool ()
  4. Click "Create a new simulation from selection" to proceed to the next step

Note:

There are 4 high-resolution image tiles per LiDAR tile -- the more image tiles you choose, the longer things will take to process.
